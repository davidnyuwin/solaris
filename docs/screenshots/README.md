# Solaris UI Screenshots 📸

Place your high-resolution macOS application screenshots in this directory. 

To maintain the polished visual standard of the project:
*   Ensure dark mode is active to complement the translucent glassmorphism aesthetics.
*   Capture the central Siri-style abstract orb in active states.
*   Save the screenshots using the following exact filenames to match the links in the main `README.md`:
    *   `dashboard.png`: Central control dashboard with the visual orb and diagnostic sweep card.
    *   `local-diagnostics.png`: Active logs view showing live parsed logging traces.
    *   `settings.png`: App settings view showing the connection mode picker (Mock Mode, Experimental REST Mode, Local Diagnostics Mode).
